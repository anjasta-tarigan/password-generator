# Password Generator

A modern, client-side password generator built with React + Vite. Generate strong, secure passwords instantly with customizable options.

## Features

- **Password length**: 4–128 characters
- **Character sets**: Uppercase, Lowercase, Numbers, Symbols
- **Exclude similar characters**: Avoid confusing chars like `I`, `l`, `1`, `O`, `0`
- **Strength meter**: Visual indicator from Weak → Very Strong
- **Copy to clipboard**: One-click copy
- **100% client-side**: No data sent anywhere — fully private
- **Glass morphism UI**: Modern dark theme with animated background

## Tech Stack

- React 19
- Vite 8
- Pure CSS (no external dependencies)

## Getting Started

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
```
